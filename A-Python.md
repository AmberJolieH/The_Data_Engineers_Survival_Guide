# Declare a variable:
## Numerical
### Whole number:
- variable_name = value
- ***Example:***
    - age = 34
    - print(age)
        - prints 34 in the console

### Decimal Point 
- ***Example:***
    - variable = 20.5
    - print(variable)
        - prints 20.5 in the console

## String
- ***Example:***
    - name = "Amber"
    - print(name)
        - prints Amber to the console
